Struktur Data di Balik Stock.n

Setiap fitur dibangun di atas konsep struktur data dan algoritma yang nyata — bukan sekadar teori.

Struktur Data

Dictionary (Hash Map)

Data inventori disimpan menggunakan Dictionary dengan kunci SKU sehingga akses dan pengelolaan data dapat dilakukan dengan cepat — kompleksitas O(1).

inventory = {"BRG001": {"nama": "...", "stok": 15}}
# Akses O(1) dengan kunci SKU
Struktur Data

Stack (LIFO)

Riwayat aktivitas disimpan sebagai Stack. Fitur Undo bekerja dengan operasi POP yang mengambil dan membalikkan aksi terakhir secara otomatis.

activity_stack.append(action) # PUSH
last = activity_stack.pop()   # POP (Undo)
Algoritma

Linear Search

Pencarian sekuensial pada seluruh data inventori, mencocokkan query dengan nama barang dan kategori hingga ditemukan hasil yang sesuai.

for sku, detail in inventory.items():
  if query in detail['nama'].lower():
    result[sku] = detail  # O(n)
Algoritma

Bubble Sort

Pengurutan data berdasarkan stok dengan membandingkan elemen berpasangan secara berulang hingga seluruh data terurut dari terkecil.

for i in range(n):
  for j in range(0, n-i-1):
    if items[j] > items[j+1]: # O(n²)
      swap(items[j], items[j+1])
